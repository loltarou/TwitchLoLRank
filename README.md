# Twitch Rank Emblem - 連携および利用ガイド

本システムは、Twitchのチャット欄にLeague of Legends（LoL）の現在のランクを表示するための拡張機能および連携システムです。
以下の手順に従って、RiotアカウントとTwitchアカウントの連携、および拡張機能の導入を行ってください。

---

## 1. アカウントの連携手順（初回登録）

拡張機能でランクを表示させるためには、まずご自身のRiotアカウントとTwitchアカウントを紐付ける必要があります。

1. **連携用Webサイトへアクセス**
   ブラウザで以下のシステムURL（トップページ）を開きます。
   `https://twitchrank-server-932555910114.asia-northeast1.run.app/`

2. **Riot IDの入力**
   画面の指示に従い、ご自身のRiot ID（ゲームネームとタグライン。例: `Name#1234`）を入力し、確認ボタンを押します。

3. **アイコン認証（本人確認）**
   システムから「アイコンを〇〇番（29番または28番）に変更してください」という指示が表示されます。
   * LoLのクライアントを起動し、指示された番号のプロフィールアイコンに変更してください。
   * アイコンを変更してしばらく待つと、システムが自動で検知して認証が完了します。

4. **Twitchアカウントとの連携**
   アイコン認証に成功すると、Twitch連携ボタンが表示されます。
   ボタンをクリックしてTwitchの認証画面へ進み、「許可」を押してアカウントを紐付けます。
   完了画面が表示されれば、データベースへの登録は成功です。

---

## 2. ブラウザ拡張機能の導入

次に、Twitchの画面上でランクを表示させるためのブラウザ拡張機能を導入します。

1. **拡張機能管理画面を開く**
   Google Chrome（またはEdge）を開き、URLバーに以下を入力して移動します。
   `chrome://extensions/`

2. **デベロッパーモードの有効化**
   画面右上にある「デベロッパー モード」のスイッチをオンにします。

3. **拡張機能の読み込み**
   画面左上に表示される「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。
   本システムの拡張機能ファイル群（`manifest.json`など）が入っているフォルダを選択します。

4. **有効化の確認**
   拡張機能の一覧に「Twitch Rank Emblem」が追加され、スイッチがオンになっていることを確認してください。

---

## 3. 動作確認

すべての設定が完了したら、正常に動作するかテストを行います。

1. ご自身のTwitchの配信画面（チャット欄が見えるページ）を開きます。
2. チャット欄にテスト用のコメントを送信します。
3. 送信したご自身の名前の横に、LoLのランク（例: `[GOLD IV]`など）が正常に表示されれば設定完了です。

---

## 4. 連携の解除手順

アカウントの紐付けを解除したい場合は、以下の手順を実行してください。

1. 連携用Webサイト（トップページ）へアクセスします。
2. 画面右上にある「認証済みの解除」ボタンをクリックします。
3. 連携時と同様にRiot IDを入力し、指示された番号へプロフィールアイコンを変更して本人確認を行います。
4. 認証が完了すると、データベースからあなたの連携データが完全に削除されます。
